# Generated code (Code Apps CLI output)

This folder is reserved for output generated by the Power Apps Code Apps CLI —
typically Dataverse table models and services generated in **Phase 3**.

## Rules

- **Never manually edit generated files.** They are overwritten on regeneration.
- **Never import generated services directly** from handwritten feature or shared code.
  Wrap them behind the repository/adapter interfaces in each feature's `data/` folder
  (e.g., `DataverseAssetRepository` implementing `AssetRepository`).
- Some generators emit output **outside** this folder (e.g., Power Automate flows
  currently generate into `src/services`, `src/models`, and `schemas/logicflows`).
  Document any such path as generated and apply the same no-manual-edit rule.

In Phase 1 this folder intentionally contains only this README.
